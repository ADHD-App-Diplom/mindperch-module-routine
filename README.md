# Routine Module (`mindperch-module-routine`)

## High-Level Overview
The **Routine Module** helps users automate the recurring "skeleton" of their day. It breaks down complex routines (like Morning or Bedtime) into sequential, timed steps.

## Key Features
- **Routine Player**: An interactive guide that leads the user through each step with dedicated timers and progress indicators.
- **Routine Builder**: Easy management and reordering of steps via drag-and-drop.
- **Start-Friction Support**: Designed to help users overcome the initial hurdle of starting their day.

## Technical Details
- **Persistence**: Uses `IsarRoutineRepository`.
- **Sync**: Compatible with PerchLink for local cross-device synchronization.
